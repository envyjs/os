What's new in Envy

Envy 8.0, the newest release of Envy, is finally here. We've added a lot of updates for you, from right here in the Envy Store to all around the rest of Envy. We hope you enjoy these new features!

What's new:

Envy 8 is not only big on features, but is also a massive cleanup release. Bugs have been fixed all around and there's a new clean UI for you to enjoy!
  • Redesigned Launcher menu
  • All new Envy Store
  • Settings redesign
  • Browser and Aspen are now split
  • Pretty UI tweaks and changes that make Envy feel nicer to use
